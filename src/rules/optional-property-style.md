# optional-property-style

Enforces unified style for optional properties.

## eslintrc.js

```ts
"@skylib/disallow-identifier": [
  "error",
  {
    style: "combined" | "optional" | "undefined"
  }
]
```

## Options

| Name | Description |
| :------ | :------ |
| `style` | Optional property's style. |
